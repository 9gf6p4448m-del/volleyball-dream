## 教訓（本輪新增）

1. **體感問題二度回報＝第一次調查漏了層**：攔網「像接球」首查只驗 sim 層（tryBlock 正確），漏了輸入層緩衝變質——輸入層與 sim 層要分開驗。
2. **探針先於判斷**：接噴「感覺沒看到」→ 探針量出備援誤觸發 19%＋救噴 0 次＋真噴結構性不存在——三個真相全非直覺可得。
3. **死碼考古**：玩家心智模型（自動攔網）與實況（全手動）背離兩週無人發現——清死碼時要問「玩家知道這功能沒了嗎」。
4. **「分配到」≠「執行到」的姊妹版**：「機制在」≠「實戰會發生」（救噴魚躍窗口×擲骰＝實戰 0 次）。

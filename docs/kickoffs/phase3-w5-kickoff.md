## 討論題（七題；每題附現況與我的推薦，Sawmah 拍板）

### Q1 賽季輪迴的保留/重置清單

草稿設計：季末（奪冠或止步）→「進入下一屆」按鈕。

| 保留（帶著走） | 重置（新一屆） |
|---|---|
| 主角屬性/技巧/成長點數 | 賽程（同六場模板） |
| 名冊隊友成長 | 戰績（0 勝 0 敗重來） |
| 招募進度與已入隊成員 | 每場比賽種子（決定論衍生新種子，不重演上一屆） |
| 宿敵記憶（scouting 跨屆累積） | |
| 已播劇情（不重播 teach-*/debut 等） | |
| 先發編排（lineup/trust） | |

推薦照此清單。要調整的點請圈出。

### Q2 難度模型（Sawmah 已提出的顧慮：「止步的人還要打更強的隊，不合理」）

**草稿方案＝難度綁成就，不綁屆數**：
- 止步 → 下一屆對手**維持原強度**。變強的是你（名冊成長＋招募＋技巧），失敗的回報是「帶著成長捲土重來」。
- 奪冠 → titles+1 → 衛冕屆對手全屬性 +3×titles。敘事：「全國都在研究衛冕軍」——與宿敵情蒐天然呼應。

替代案（比較用）：
- B. 每屆固定微升（不分輸贏）——就是 Sawmah 覺得不合理的那種，列出供對照。
- C. 學年制新老交替（三年級畢業/新生入學，各隊陣容浮動可強可弱）——最真實但敘事/命名工程量大，可當 Phase 4+ 遠期方向。

推薦 A（草稿已實作）；若選 A，衛冕升級幅度（現 +3/冠）可討論。

### Q3 賽中換人／輪轉替補要不要進 W5？

7.7 `cancelFaultPoints` 的真實觸發源（W3 定死的純註解接線點）。原排 W5/W6。
- 進 W5：規則完整度高，但工程大（換人 UI＋sim 輪轉驗證啟用＋決定論影響）。
- 留 W6+：W5 聚焦輪迴，換人獨立成週。
推薦**留 W6+**（輪迴已解決玩家實際卡點，換人無人在等）。

### Q4 名冊滿的釋出機制要不要進 W5？

W4 拍板「無空位＝條件保持、progress 不清，W5 若做釋出機制再接」。名冊 10 人、
招募候選 5 人、空位 3——招滿後想收第 4 人需要「請走某人」。
推薦 **W5 先不做**（實際要滿編至少兩三屆後才會遇到），排 Phase 3 收尾或 Phase 4。

### Q5 新一屆的劇情量級

現況：teach-*/debut 等已播不重播（正確）；第二屆＝零對話直接開打。
- 最小：衛冕開場一段＋捲土重來開場一段（兩段對話，隊長講話）。
- 中量：另加宿敵跨屆差分（「又是曜石——第三年了」）。
- 零劇情：先不加，命名工程時一併寫。
推薦**最小**（兩段開場對話，與命名工程一起寫也可）。

### Q6 招募節奏確認

跨屆累積下的實際達成屆數：曜石（一屆遇 2 次）＝最快 1 屆；天鷹＝決賽勝 1 屆；
白浪/鐵霧（勝 2）＝至少 2 屆；北原（勝 3）＝至少 3 屆。
這個節奏（全招滿大約要 3 屆）OK 嗎？要調就是改 RECRUIT_CONDS 的 wins 數字（一行）。

### Q7 棄賽 UX（Sawmah 實測踩雷：四場 0:25）

中途離開比賽＝記棄賽敗 0:25（拍板的防 reload 白嫖機制）。實測發現：測試/誤觸離開
也會吃敗場，一屆四場 0:25 全滅的體驗很差。
- 推薦：**離開前加確認彈窗**「中途離開將記棄賽敗（0:25）——確定離開？」機制不變。
- 替代：提供每屆 1 次「無代價棄賽」額度（機制複雜化，不推薦）。

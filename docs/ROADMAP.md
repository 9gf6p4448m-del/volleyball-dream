# 排球夢 — 開發路線圖 v1（2026-07-21）

統一驗收：手機穩定60 FPS硬底線（120Hz裝置不鎖幀、120為餘裕目標非通過條件）、模擬固定步長與幀率脫鉤、觸控與滑鼠雙輸入皆可玩、模擬核心與畫面分離不破例。

## Phase 0 — 基準測試（定生死）
12個寫實比例動畫模型（Mixamo系）＋標準球場＋物理球＋FPS顯示，部署後在Sawmah手機實測。
- 通過（穩定60；120Hz裝置另記錄不鎖幀數值）→ 寫實美術定案
- 未通過 → 逐項降規（模型面數/材質/光影）找出可行上限，再定美術
- 附帶產出：專案骨架（Three.js + PWA + 可包殼結構）

## Phase 1 — 單場可玩（垂直切片）
- 玩家位置：**主攻手**（五種動作全碰得到，驗證手感最完整）
- 賽制：**單局25分rally point**，完整6v6規則與輪轉
- 五動作操作全套（觸控＋滑鼠統一向量邏輯）
- 混合視角切換（發/扣第一人稱，其餘第三人稱）
- 11人回合AI狀態機（本階段地基級任務，Fable）
- 資料層v1：屬性＋技術＋身高系統的資料結構定型（地基級）；身高設計為時變屬性、預留信任參數（為Phase 3成長期身高與信任系統鋪路）
- **視線欺敵原型**：第一人稱扣球的視線方向餵給攔網AI，看左打右可行
- 球影、鏡頭可讀性、擊球音效（全手動走位的公平前提）

## Phase 2 — 位置即職業
舉球員 → 自由球員 → 攔中 → 對角逐一開放，各自獨立操作體系與教學。自由球員規則特例（不可攻擊、不轉前排）單獨處理。

## Phase 3 — 高中第一賽季（第一個劇情章節）
創角（真實身高輸入＋位置建議）、**成長期身高曲線**（三年間身高隨賽季成長，位置認同的敘事線）、訓練循環（屬性/技術成長）、**信任即戰術**（舉球員AI分配由劇情與表現的信任值驅動）、劇情主線＋位置差分支線、高中聯賽賽程與關鍵戰。

## Phase 4 — 生涯延伸
聯賽→職業的後續章節、架空聯賽金字塔全貌、選秀/升降、對手球隊與名將群像。

## Phase 5 — 多人連線（到時專門討論）
6真人一隊線上對戰。前提已備：純模擬核心、位置專精養成。新戰場：後端伺服器、同步方案、配對。

## Phase 6 — 上架（最後討論）
Google Play（TWA）→ App Store（Capacitor包殼，注意Apple對網頁包殼的審核要求）。

## 已定案決策清單（討論不重開）
1. 主角球員模式，不做整隊控制
2. 位置即職業，資料層第一天建對
3. 真實身高輸入，真實影響一切
4. 混合視角自動切換（試玩可調參，不改方向）
5. 走位全手動無提示，真實線索（球影等）補足
6. 成長 = 屬性＋技術兩層
7. 世界觀 = 高中→聯賽→職業，架空聯賽金字塔
8. 主線位置無關、支線位置差分
9. Phase 1 = 主攻手、單局25分
10. 模擬核心與畫面分離（多人前提）
